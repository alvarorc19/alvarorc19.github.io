Ising model

Defined by Hamiltonian\ $$H = E = -J \sum_{\langle i j \rangle}s_i s_j - h \sum_i s_i.$$

Simulation steps

  1. Make a parser and have hold the relevant data for the simulation

  3. Run simulation i. Measure current state ii. Perform algorithm iii. Write new state (tell how verbose in an option, also log) iv. Compute observables or measurables (also tell if do this in config)
  4. Quit simulation and end program
