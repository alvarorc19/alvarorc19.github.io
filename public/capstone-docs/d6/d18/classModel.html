Detailed Description

template<typename T>
class Model< T >

Base clase for different models.

This class provides the core functionality for simulating statistical mechanical models (e.g., Ising, Potts, XY models) on lattices. It handles energy computations, partition functions, and state management.

Template Parameters
Tdata type used for lattice sites
